Gambler's Ruin — Markov Chain Calculator

An interactive study of the Gambler's Ruin problem using Discrete Time Markov Chains. Four progressively complex strategies, each computed analytically or via linear system solving.

Parameters
About

Bet exactly $1 each round. Game ends at $0 (ruin) or $N (goal). Win probability solved via closed-form Markov recurrence. Expected duration D(k) = [k − N·W(k)] / (q − p).
